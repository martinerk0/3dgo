This is root directory of 3DGo application.
You can start the application by:
1. install node.js from:     https://nodejs.org/en/
2. run command:              node app.js <number of port>   (e.g. node app.js 3000 )
3. connect with browser on:  http://localhost:3000/         (Currently supported are Chrome and Safari)


Currently supported browsers are: Chrome, Safari, (and Firefox too, but you have to change one flag, see text of the thesis.)


JSDoc documentation can be generated by running command from this directory:
1. jsdoc -r . -c _JSDoc/jsdoc_conf.json _JSDoc/README.md

It generates html files into directory out



If you want to add new board from Blender, add it to /serverSide/boards
